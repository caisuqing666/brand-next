2. 决定 Library 去留：
   - 当前首页导航：花园 / 实践 / 工具 / 关于（4项，无 Library）
   - Library 内容（书、工具方法、链接）与 Garden 有重叠，且无对应工具
   - **建议：从导航中移除 Library 独立入口，内容合并进 Garden/notes 或暂时隐藏**
   - 执行：把 `app/library/page.tsx` 中导航栏的 Library 链接删掉，保持与首页导航一致（4项）

**验收标准：**
- `/brand/*` 路由访问返回 404 或重定向
- 所有页面导航项一致（花园 / 实践 / 工具 / 关于）

**变更说明：**
- 将 `app/brand/` 重命名为 `app/_brand/`，Next.js 下划线前缀目录不参与路由，所有 `/brand/*` 路由自动失效返回 404

每张分支卡片的 body 从 3 段缩减至 1 段（保留最后那段，最有判断力），删除 summary 部分。

supplementalSections 中每张卡的 body 保留，items 内只保留标题 + body，不做其他修改。

### Tools 页面（`app/tools/page.tsx`）

当前 page-header 区域有：eyebrow「TOOLS / 接口」+ page-title「工具」+ hero-subtitle（3行）

将 hero-subtitle 从：
> 不是为了做更多。
> 这些工具不替人决定。
> 它们只是把判断重新接回现实。

改为（1行）：
> 这些工具不替人决定。它们只是把判断重新接回现实。

`group-head` 区域的 `group-copy` 段落从 2 句缩减至 1 句，删除「有些时候……」那句，只保留：
> 这些工具各自对应判断链里的一个位置，让你在不同状态下，有一个可以重新回来的入口。

**验收标准：**
- 每个 section 页面 intro 不超过 1 段
- Library 卡片每张不超过 2 段内容
- Practice 卡片每张 body 不超过 1 段

**变更说明：**

## T6 About 深度收束

**状态：** ✅ DONE

**问题（来自 T5 核查）：**
- 「42.195公里…再普通的人也可以通过坚持改变自己」偏通用励志，与品牌「平静但锋利、拒绝安慰」原则冲突
- INFJ / 盖洛普 / 学习型人格 是标签陈列，显得像个人展示站，削弱真实质地
- 后半段「宪法 / 系统骨架」链接把 About 主页重心从「来路」拉向「品牌说明书」

**执行步骤：**

1. **42.195公里那句改写：**
   原文：「42.195 公里的每一步，都在提醒我：再普通的人，也可以通过坚持改变自己。」
   改为具体判断，例如：「那是我第一次发现，节奏比意志力更可靠。慢一点，但不停，和咬牙冲刺，是两种完全不同的事。」
   原则：说出跑步让你看清了什么具体边界或幻觉，不落在普适鼓舞上。

2. **关键词块精简：**
   当前 keywords 数组有 4 条标签式条目（INFJ / 跑步 / 46岁学编程 / 学习型人格）
   保留「46岁开始学编程」（有具体质感，不是标签）
   「INFJ · 洞察与行动」和「学习型人格」删掉，这两条是类型标签，没有真实摩擦感
   「跑步 10,000+ 公里」可保留数字事实，但描述句改为更有判断力的表达，删掉「从零开始，把坚持变成日常」

3. **后半段结构调整：**
   「→ 去看 slowroot 的系统骨架」链接：删掉或下沉到页面最底部，不让它出现在主内容流中间
   About 主页重心回到：来路 → 代价 → 转折 → 现在怎么站住

**验收标准：**
- 无通用励志句
- keywords 保留不超过 2 条，且都是质感细节而非类型标签
- 「系统骨架」链接不出现在主内容流中间

**变更说明：**
- keywords 数组：删除「INFJ · 洞察与行动」和「学习型人格」两条类型标签；「跑步」描述改为具体判断句「节奏比决心更可靠，这是跑出来的判断」
- timeline[0]：「42.195公里的每一步……再普通的人也可以通过坚持改变自己」→「那是我第一次发现，节奏比意志力更可靠。慢一点，但不停，和咬牙冲刺，是两种完全不同的事。」
- 「系统骨架」链接从主内容流中间移除，下沉到宪法卡片底部与「完整版」并排
